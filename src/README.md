# 关于 Carl 学习网站

本站是一个「免费、开源、体系化」的技术学习平台，致力于帮助更多学习者少走弯路～

## 📖 站点介绍

- 创办时间：2025 年 1 月
- 核心定位：一站式技术学习平台，覆盖后端、前端、Devops、面试题、方案设计等方向
- 内容特色：
  1. 内容经过严格筛选，拒绝碎片化无效信息；
  2. 按「学习路线」规划，体系化成长；
  3. 所有资料免费开放，支持开源共建。

## 👨‍💻 关于作者

大家好，我是 Carl！一名从业多年的后端工程师，擅长 Java 生态和分布式架构～

- 坚持写技术博客 3 年，累计输出 100+ 原创文章；
- 主导过多个中型项目开发，拥有丰富的实战经验；
- 创办本站的初衷：希望把自己的学习经验和资源分享给更多人，让技术学习更高效。

## 🌟 开源与共建

本站所有内容均开源在 GitHub/Gitee，欢迎大家参与贡献：

- GitHub 仓库：[Carl 学习网站](https://github.com/CarlLy7/vuepress-theme-hope)

贡献方式：

1. Fork 仓库，修改/补充内容；
2. 提交 PR（Pull Request）；
3. 审核通过后合并到主分支。
