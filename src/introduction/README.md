# 网站介绍

欢迎来到 Carl 学习网站！这是一个「免费、开源、体系化」的技术学习平台，致力于帮助更多学习者少走弯路。

## 🎯 网站定位

本站是一个一站式技术学习平台，覆盖后端开发、DevOps、面试题、系统设计等多个技术方向，为学习者提供高质量、体系化的学习资源。

## 📖 站点介绍

- **创办时间**：2025 年 1 月
- **核心定位**：一站式技术学习平台，覆盖后端开发、DevOps、面试题、系统设计等方向
- **内容特色**：
  1. 内容经过严格筛选，拒绝碎片化无效信息
  2. 按「学习路线」规划，体系化成长
  3. 所有资料免费开放，支持开源共建

## 📚 内容体系

### 面试题库

精选高频面试题，涵盖 AI、后端、场景设计等多个方向，助你轻松应对技术面试。

- **AI 面试题**：RAG 技术、幻觉问题解决方案等
- **后端面试题**：日常搜集的高质量面试题
- **场景设计题库**：秒杀系统、短链系统、配置中心、feed 流系统、乘车系统、分布式事务、幂等性、数据一致性、注册用户名唯一性、数据迁移、站内消息、UV 统计、缓存一致性、订单超卖等 14+经典系统设计方案

### 消息中间件

深入解析 Kafka、Disruptor 等消息队列，掌握消息中间件核心原理与实战应用。

- Kafka 深度解析
- Disruptor 高性能队列
- 消息队列经典问题解决方案
- 消息中间件技术选型与对比

### DevOps

Kubernetes、Jenkins 等容器化与 CI/CD 实践，提升运维效率与系统稳定性。

- Kubernetes 学习笔记
- 使用 kubeadm 安装 k8s 集群
- Jenkins + Kubernetes 实现 CI/CD

### 程序员进阶之路

JVM 调优、SQL 优化、系统设计等进阶内容，助力技术能力持续提升。

- JVM 参数通用模板与调优
- SQL 优化手段
- MySQL 高性能批量插入优化
- 事件驱动模式进行业务解耦
- 系统设计常识与原则和系统性能优化原则
- 线上故障分析
- 线程池参数应该如何设置

### AI 应用开发

提示词工程、RAG 技术等 AI 应用开发知识，紧跟技术前沿。

- 提示词工程
- RAG 相关技术

### 项目实战

真实项目案例与经验分享，理论结合实践，快速积累项目经验。

- 虚拟货币交易系统笔记
